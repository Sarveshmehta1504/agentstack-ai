# AgentStack Roadmap 🗺️

Our goal is to build the category-defining developer platform for visual multi-agent workflows.

## Phase 1: Foundational Visual OS (Current)
*   [x] Dark futuristic shell design system
*   [x] Interactive modular views switcher
*   [x] SVG-backed Node graph canvas layout
*   [x] Zustand-backed active run triggers
*   [x] Unified terminalActivity telemetry logs
*   [x] Memory Vector explorer filter

## Phase 2: Live Integration Bridges
*   [ ] OAuth integration for GitHub, Vercel, Supabase
*   [ ] Local system terminal execution stream
*   [ ] Real SQLite local storage for workflow persistence
*   [ ] Semantic indexing using Transformers.js

## Phase 3: Developer Ecosystem
*   [ ] AgentStack Plugin Marketplace
*   [ ] Cloud Orchestration Engine (hosted workflows)
*   [ ] Interactive CLI node designer (`agentstack-cli`)
