# Contributing to AgentStack 🤝

We welcome all developers to join and build the future visual agent operating system!

## How to Contribute

1.  **Fork the repository:** Click the "Fork" button on the GitHub repository.
2.  **Create a feature branch:**
    ```bash
    git checkout -b feat/my-new-feature
    ```
3.  **Implement your changes:** Ensure code matches our layout guidelines, clean TypeScript interfaces, and dark aesthetic components.
4.  **Format and lint:**
    ```bash
    npm run lint
    ```
5.  **Submit a PR:** Push your branch and open a Pull Request outlining your changes.
