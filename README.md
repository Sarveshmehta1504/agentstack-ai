> **Open-Source Visual AI Workflow Operating System for Developers.**

AgentStack is a developer-first AI orchestration platform designed to coordinate autonomous agent meshes, automate complex engineering workflows, query persistent semantic memory, and monitor pipeline execution traces via a futuristic terminal shell.

---

## 🌟 Key Features

*   **Visual Workflow Canvas:** Drag, drop, and connect AI agents, database triggers, terminal commands, and API integrations.
*   **Multi-Agent Mesh:** Orchestrate team-based agent roles (Architect, Frontend, Backend, Review, Debug) working collaboratively.
*   **Vector Memory Explorer:** Map, query, and trace semantic vector memories and procedural links.
*   **Live Telemetry Shell:** Production-grade interactive terminal stream monitoring local pipeline activity.
*   **GitHub & Service Integrations:** Seamlessly hook into GitHub, Vercel, Supabase, and Slack automation.

---

## 🛠️ Tech Stack

*   **Framework:** Next.js 15 (App Router, Turbopack)
*   **Styles:** Tailwind CSS v4 & shadcn/ui
*   **Animations:** Framer Motion
*   **State Management:** Zustand
*   **Developer UI:** Lucide Icons & Custom SVG Graph Canvas

---

## 🚀 Quick Start

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/agentstack.git
    cd agentstack
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run development server:**
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:3000` to access the visual OS workspace.
